Installation Instructions

Follow the steps below to install Tic Track on your system.

macOS

Download the `.dmg` file, open it, and drag Tic Track into your Applications folder. Warning : macOS may block the installation. Make sure to watch this video to install correctly.

Windows

Download the `.exe` file and double-click to start the installation.

Linux

Download the `.AppImage` file, make it executable, and run it.
